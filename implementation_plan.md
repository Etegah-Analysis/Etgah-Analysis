# استبدال شارت TradingView بشارت مخصص لمؤشر تاسي

نظراً لأن TradingView تمنع عرض مؤشر السوق السعودي "تاسي" في الشارت الخارجي، سنقوم ببناء شارت مخصص بالكامل داخل المنصة ليعرض بيانات تاسي.

## الخطة المقترحة

### 1. بناء شارت مخصص (Custom Chart)
سنقوم بإعادة استخدام مكتبة `lightweight-charts` (التي استخدمناها في بداية المشروع) لبناء رسم بياني احترافي يتناسب مع تصميم المنصة المظلم (Dark Mode).

### 2. جلب البيانات (Data Fetching)
سنقوم ببرمجة دالة (Function) تحاول جلب بيانات "تاسي" الحقيقية من واجهات برمجة عامة (مثل Yahoo Finance عبر رابط وسيط Proxy). 

### 3. نظام الطوارئ (Fallback System)
نظراً لأن الواجهات المجانية غالباً ما تتعرض للحظر أو التوقف، سأقوم ببرمجة نظام ذكي يولد بيانات حركة (شموع يابانية) تحاكي حركة مؤشر تاسي الحقيقية (حول مستويات 11,000 إلى 12,000 نقطة) لفترة سنة كاملة. 
بهذه الطريقة، **لن يتعطل الشارت أبداً**، وسيظهر دائماً بشكل احترافي للمستخدمين حتى لو توقف مزود البيانات.

## [!IMPORTANT] User Review Required
هل توافق على هذا الحل؟ استخدام شارت مخصص مع نظام بيانات احتياطية يحاكي حركة السوق في حال توقف الـ API المجاني؟
